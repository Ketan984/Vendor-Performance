# Vendor-Performance
This project is an  Data Analytics solution built to analyze vendor and brand performance using SQL, Python, and Power BI.  The dashboard provides actionable insights into sales performance, profitability, stock turnover, purchase contribution, and unsold capital to help businesses make data-driven decisions.
Overview

This project focuses on analyzing vendor performance using Python, MySQL, and Power BI.

The objective is to evaluate vendor efficiency, delivery performance, pricing patterns, and overall contribution to business operations. The project follows a complete data analytics workflow — from data loading and cleaning to SQL analysis and dashboard visualization.

This project demonstrates practical skills in:

Data Cleaning & Preprocessing

Exploratory Data Analysis (EDA)

SQL Querying & Business Analysis

Data Visualization & Dashboard Development

Business Insights Generation

📁 Dataset

The dataset contains vendor-related transactional.
The data was loaded and processed using Python before being stored in MySQL for structured querying.

Cost variations

Defect rates

Cost comparison across vendors

Monthly performance trends

JOINs

GROUP BY

Aggregate functions

Subqueries

📈 Vendor Performance Score
